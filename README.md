## 🚀 Featured Projects

**[`cs.muet.edu.pk`](https://cs.muet.edu.pk)** | `Next.js 15` · `TypeScript` · `Drizzle ORM` · `Neon PostgreSQL` · `Vercel`

> Official production system serving students, faculty, alumni, industry partners, and accreditation bodies.

- **25-table normalized schema** across 6 modules; JSONB SDG tagging; atomic delete+audit transactions
- Strategic JSONB denormalization for read-hot tables minimizes JOIN overhead; strict 3NF for all transactional/security tables

### 🖥️ KernelViz OS — Operating System Simulator
**[`Source`](https://github.com/umerqureshi409)** | `C (gcc shared lib)` · `Python` · `WebSocket` · `HTML/CSS/JS Canvas` · `psutil` · `ctypes FFI`

> CEP submission for CS-261 OS, covering all 9 accreditation characteristics.

- **C core (`libkernelviz.so`):** Round Robin & Priority schedulers, First-Fit allocator with block coalescing, inode-based VFS, pipes, message queues, DFS-based deadlock detection
- **Python bridge** merges simulation with live `/proc/sys` data via `psutil`; WebSocket at 1 Hz
- **10-tab browser dashboard:** Gantt chart, memory map, process Kill table, Real Stats — pure DOM/Canvas, no frameworks
- **Dual-mode:** LIVE+SIM on Linux; REAL-ONLY (psutil only) on Windows/Mac
